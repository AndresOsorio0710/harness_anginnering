# ENGINES.md — Índice de features completadas

> Este archivo vive en la **rama de la feature** (`feature/{NNN}_{name}`).
> Es actualizado por el `leader` al cerrar una feature.
> **No editar manualmente.**
>
> Cada entrada registra una feature completada: el nombre, la rama,
> el commit SHA y la fecha de cierre.
>
> Cuando el humano solicite integración a la rama original (main/master/dev),
> este archivo sirve como registro de qué features están listas para fusionar.
